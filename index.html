The number of video productions are increasing. This webpage will serve as a resource to produce high quality content with consistent branding. Standards for splash screens, lower thirds, discalmer and logos are going to be avaliable for download. When creating videos please use the appropriate standards as it pertains to the respective auxilary organzations (National, MKAC, Atfal, Ansar and Lajna events).

 

Please use the checklist below before the start of the event.

Splash Screen

  • The splash screen should have the logo downloaded from Logo Section.
  • Event name / Date / Time should be included.
  • Appropriate font size should be tested prior to event.
  • Refrain from using bright font/background colours.
  • When applicable please use template provided in link below.
